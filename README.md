
# Divine Routine – Personal Care Android App

An Android application for **haircare and skincare recommendations**, combining:
- Personalized routines  
- Allergen-aware product filtering (haircare only)  
- NLP features (Text‑to‑Speech & Speech‑to‑Text)  
- Camera-based before/after tracking  
- SQLite-based local storage  
- Ayurvedic + Cosmetic hybrid approach  

This repository contains the complete source code of the Divine Routine app.

---

## Features

### Haircare Recommendations
- Select hair type, texture, and concerns  
- Allergen filtering (parabens, sulfates, silicones, alcohols)  
- Product suggestions + care routine  

### Skincare Recommendations
- Select skin type (oily, dry, normal, combo)  
- Concern-based product suggestions  
- Morning + night routine suggestions  

### NLP Features
- **Speech‑to‑Text (STT)** → fill forms hands‑free  
- **Text‑to‑Speech (TTS)** → read routines aloud for accessibility  

### Before / After Tracker
- Capture images and store progress  
- Review improvement over time  

### SQLite Database
- Stores user history  
- Routine decisions  
- Product metadata

## ▶️ Running the App

1. Open Android Studio  
2. Select **Open Project**  
3. Choose the folder:  
   `divine-routine-personal-care-android-app/`  
4. Let Gradle sync  
5. Click **Run ▶**
